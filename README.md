# Yupi

Yupi — это одностраничный прототип сайта знакомств, сделанный на чистом HTML, CSS и JavaScript.

## Что умеет
- Регистрация профиля
- Просмотр анкет
- Лайки
- Имитация взаимного совпадения
- Чат после взаимного лайка
- Сохранение данных в `localStorage`

## Как запустить
1. Открой `index.html` в браузере
2. Или опубликуй проект через GitHub Pages

## Структура
- `index.html` — основной файл сайта
- `README.md` — описание проекта

## Примечание
Это демо без сервера и базы данных. Все данные сохраняются только в браузере пользователя.
